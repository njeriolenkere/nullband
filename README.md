# nullband
nullband is midi interfecae like garageband. You can develop it further to add drums and piano. Simply run the code via live server connect your midi controller then <!--Click anywhere on the page to activate audio (for browsers that require user interaction).--> and begin to play.

To learn the basics, read this: https://webmidijs.org/docs/
