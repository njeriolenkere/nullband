# nullband
nullband is midi interface website like garageband. You can develop it further to add drums, piano and more instruments. Simply run the code via live server (if you are using visual studio code ide) then connect your midi controller then Click anywhere on the page to activate audio (for browsers that require user interaction). and begin to play on your midi controller.
Watch this video to see the results: https://youtube.com/shorts/v7Y2vduo6es?si=3eS9mOshjWFdr0AK

To learn the basics, read this: https://webmidijs.org/docs/
